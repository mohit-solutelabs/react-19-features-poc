# About this repo

This repo provides different POCs of React v19 features for developers and stackholders.

## About the Setup

This project was boostrapped from scratch using Vite, React v19, tailwindcss using TypeScript.

## Setup in local

### Prerequisites

- Node v18+
- Yarn v1.22+

### Steps

1. Clone the repo
2. Run `yarn install`
3. Run `yarn dev`
